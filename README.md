# Azure Secure Landing Zone

A secure, enterprise-style Azure network topology built from scratch using
Infrastructure as Code (Bicep). No public IPs exposed — all VM access via
Azure Bastion through a hub-and-spoke VNet design.

## Status
In Progress

## Planned Azure Services
- Azure Virtual Networks (Hub + Spoke)
- VNet Peering
- Network Security Groups (NSGs)
- Azure Bastion
- Azure Virtual Machines (Linux)
- Azure Monitor + Log Analytics
